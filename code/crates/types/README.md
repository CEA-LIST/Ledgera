


# Ledgera Internal API

Defines:
- [Ledgera computation specifications](https://docs.ledgera.tech/docs/versions/v_0_1/1_concepts/#computation-specification) 
- all the internal messages that are exchanged between Ledgera nodes as part of [the protocol](https://docs.ledgera.tech/docs/versions/v_0_1/2_spec/)
- and how to validate the content of these messages w.r.t. the PKI information
  (public keys of known participants and Byzantine threshold)
